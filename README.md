# Restaurant-Management-C-
Backend server built using C++ using Crow and queue data structure to create a restaurant management portal.
This project is a simple restaurant order management backend built using C++ and the Crow web framework. It demonstrates the use of the Queue data structure in a real-world use case where restaurant orders are processed based on priority and order type.


# Features:
Place new restaurant orders
Automatically assigns priority based on order type
Fetch the next order to prepare
Mark orders as completed
View all queue states in real-time
Clear completed orders
Fully in-memory (no database)
